Context-Aware Recommendation Engine Manual
=================================================
In this manual the usage, installation and configuration of the
manual of the context-aware recommendation engine is explained. The
focus of this manual lies in the deployment of the system and ensuring
that it can be extended and not for the software design. A detailed
explanation of the software design can be found in the project
report.
